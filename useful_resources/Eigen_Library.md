# Eigen Linear Algebra Library for C++

[Eigen](https://eigen.tuxfamily.org/index.php?title=Main_Page) is a C++ template library for linear algebra: matrices, vectors, numerical solvers and related algorithms. It serves the same purpose NumPy does for Python. This library is composed entirely of header files and therefore can simply be downloaded and included directly into your source files, however, it is recommended to install Eigen using CMake according to the instructions provided in the `INSTALL` file found in the top level directory of the downloaded library.
