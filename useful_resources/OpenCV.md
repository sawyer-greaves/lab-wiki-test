# OpenCV Computer Vision Library

[OpenCV](https://opencv.org/) is the most widely used computer vision library. It has interfaces for C++, Python, and Java. The OpenCV library is composed of several modules that each contain functions and classes for performing various computer vision related tasks.

OpenCV is not a complete computer vision workflow solution. The [RoboticsFramework](https://bitbucket.org/mmrobotics/roboticsframework/src/master/) repository contains functionality built on top of OpenCV to make performing computer vision tasks even more straightforward.

---
## Installation

Installation for OpenCV is outlined [here](https://docs.opencv.org/4.x/d0/d3d/tutorial_general_install.html).

### Linux

It is recommended that OpenCV be installed on Linux using the Ubuntu repositories. This includes the `opencv_contrib` modules.
```
sudo apt install libopencv-dev
```

If you need a different version, you'll need to build from source by cloning the OpenCV repository and following the instructions in the *Introduction to OpenCV* tutorial from the OpenCV Documentation.

### Windows

It is recommended to use the prebuilt binaries offered for Windows. If you need the `opencv_contrib` modules, you will need to build from source or perhaps use a third-party package manager like conda or NuGet.
