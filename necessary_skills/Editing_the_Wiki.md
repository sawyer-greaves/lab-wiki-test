# How to Edit the Wiki

The [Bitbucket documentation](https://confluence.atlassian.com/x/FA4zDQ) has more
information about using a wiki.

The wiki itself is actually a git repository, which means you can clone it, edit it
locally/offline, add images or any other file type, and push it back to us. It will
be live immediately.

Go ahead and try:

```
$ git clone https://bitbucket.org/utahtelerobotics/lab-wiki.git/wiki
```

Wiki pages are normal files, with the .md extension. You can edit them locally, as
well as creating new ones.
